# Hospital Management System

## Overview

A web-based hospital management system to streamline patient care, appointment scheduling, and doctor-patient interaction.

## Features

*   **User Authentication:** Secure signup and login for patients.
*   **SPA Navigation:** A single-page application with routing between different pages.
*   **Form Validation:** Client-side validation for all forms.
*   **Redirects:** Appropriate redirects after signup and login.

## Tech Stack

*   **Frontend:** React, React Router
*   **Styling:** Material-UI, CSS
